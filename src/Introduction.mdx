# Facts Ark UI Components

Welcome to the **Facts Ark** component library! This is a comprehensive collection of Vue 3 components built with:

- **[Ark UI](https://ark-ui.com/)** - Headless UI components for accessibility and flexibility
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Modern utility-first CSS framework
- **[Tailwind Variants](https://www.tailwind-variants.org/)** - Type-safe variant system
- **TypeScript** - Full type safety
- **Vite** - Lightning-fast development

## Component Architecture

All components follow a consistent structure:

- **`ComponentName.vue`** - The main component file
- **`component-name.types.ts`** - TypeScript type definitions
- **`component-name.variants.ts`** - Tailwind Variants styling
- **`ComponentName.stories.ts`** - Storybook stories (documentation & testing)

## Available Components

### Form Components

- **Button** - Action buttons with multiple variants
- **Input** - Text input fields
- **Textarea** - Multi-line text input
- **Select** - Dropdown selection
- **Checkbox** - Toggle checkboxes
- **Switch** - Toggle switches
- **Field** - Form field wrapper with label and error handling
- **Fieldset** - Group related form fields

### Data Display

- **Avatar** - User profile images with fallback initials
- **Badge** - Status and category labels
- **Card** - Content containers
- **Tabs** - Tabbed interfaces

## Getting Started

Browse the components in the sidebar to explore:

- **Interactive controls** to test different props
- **Multiple story variations** showing different use cases
- **Auto-generated documentation** from TypeScript types
- **Dark mode support** via the toolbar

## Design Philosophy

This library emphasizes:

- **Accessibility first** - Built on Ark UI's accessible primitives
- **Flexibility** - Customize via props, variants, and CSS classes
- **Type safety** - Full TypeScript support
- **Consistency** - Unified API across all components
- **Performance** - Optimized for production use

Start exploring by selecting a component from the sidebar!
